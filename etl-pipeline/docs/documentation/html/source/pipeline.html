pipeline.agent_input_config.prepend_agent_name_to_ap_or_wl_or_aliases_key(agent_input_config)[source]

Prepend the agent name (level 1 key) to level 2 key. So the new level 2 key will be

Input: { ‘name_agent’: {‘ap’: [‘record_name’],

‘ap_aliases’: [], ‘wl’: [‘name_hit’], ‘wl_aliases’: [] }

}

Output: { ‘name_agent’: {‘name_agent_ap’: [‘record_name’],

‘name_agent_ap_aliases’: [], ‘name_agent_wl’: [‘name_hit’], ‘name_agent_wl_aliases’: [] }

}
