  • xpath (str) –

    Xpath that indicates the element. The xpath must not contain text()

    because return None is a requirement.

    E.g. doc.xpath(“elem”) returns a list of 2 elements,e lement_1 has a text, element_2 doesn’t. doc.xpath(“elem/text()”)` returns a list of text from element_1 only, but nothing additional information about element_2. In this case, None from element_2 is required
